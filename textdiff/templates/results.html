راهنمای آوایی (Phonetic Guide)

  • ا، آ، أ، إ، ع → a (حذف یا صدای آ کوتاه)
  • ب → b
  • پ → p
  • ت، ط → t
  • ث، س، ص → s
  • ج → j
  • چ → ch
  • ح، ه، ة، ۀ → h
  • خ → kh
  • د → d
  • ذ، ز، ض، ظ → z
  • ر → r
  • ژ → zh
  • ش → sh
  • غ، ق → gh
  • ف → f
  • ک، ك → k
  • گ → g
  • ل → l
  • م → m
  • ن → n
  • و، ؤ → v (در برخی موارد = u / o)
  • ی، ي، ى، ئ → y (در برخی موارد = i / e)
  • علائم حرکتی (َ ِ ُ ّ ْ) → حذف می‌شوند
  • ء → حذف
  • اعداد فارسی → معادل لاتین (۰→0, ۱→1, ...)
  • علائم نگارشی → حفظ می‌شوند (، . ؛ ؟ !)
