# مستندات پروژه مقایسه متن

## 📋 معرفی پروژه

این پروژه یک **وب‌اپلیکیشن پیشرفته برای مقایسه و تحلیل شباهت متون** است که به عنوان پروژه نهایی درس ساختار داده‌ها توسعه یافته است. این برنامه با استفاده از فریمورک Django ساخته شده و از الگوریتم‌های مختلفی از جمله فاصله لونشتاین (Levenshtein Distance) برای مقایسه متون فارسی استفاده می‌کند.

## 🎯 ویژگی‌های اصلی

### 1. مقایسه متن استاندارد
- استفاده از الگوریتم فاصله ویرایشی (Edit Distance)
- محاسبه دقیق تفاوت‌های بین دو متن
- نمایش عملیات‌های ویرایش (جایگزینی، درج، حذف)

### 2. مقایسه آوایی (Phonetic Comparison)
- مقایسه کلمات با تلفظ مشابه در فارسی
- تبدیل متن به نمایش آوایی
- شناسایی کلمات هم‌آوا

### 3. تجسم گرافیکی
- نمایش بصری عملیات ویرایش با استفاده از گراف
- پشتیبانی از Graphviz و D3.js
- تجسم ماتریس فاصله ویرایشی
- نمایش گراف کلمات و جملات

### 4. برجسته‌سازی تفاوت‌ها
- برجسته‌سازی تفاوت‌ها در سطح کلمه
- برجسته‌سازی تفاوت‌ها در سطح کاراکتر
- نمایش رنگ‌بندی شده تغییرات

### 5. حالت‌های مختلف الگوریتم
- **standard**: مقایسه استاندارد با فاصله لونشتاین
- **phonetic**: مقایسه آوایی (کلمات با تلفظ مشابه)
- **persian**: مقایسه با نرمال‌سازی خاص فارسی

### 6. رابط کاربری وب
- رابط کاربری ساده و کاربرپسند
- ورود دو متن و مشاهده نتایج
- نمایش جزئیات تحلیل

### 7. تولید گزارش
- تولید گزارش Markdown از نتایج مقایسه
- دانلود گزارش به صورت فایل
- شامل آمار و جزئیات کامل

### 8. پشتیبانی کامل از زبان فارسی
- پیش‌پردازش مناسب برای متون فارسی
- مدیریت علائم حرکتی (اعراب)
- نرمال‌سازی کاراکترهای فارسی

## 🛠️ تکنولوژی‌های استفاده شده

### Backend
- **Django 4.2+**: فریمورک اصلی وب‌اپلیکیشن
- **Python 3.8+**: زبان برنامه‌نویسی اصلی
- **jdatetime**: پشتیبانی از تاریخ و زمان شمسی

### Frontend
- **HTML/CSS**: ساختار و استایل صفحات
- **JavaScript**: بهبود رابط کاربری
- **D3.js**: تجسم گراف‌های تعاملی

### الگوریتم‌ها و کتابخانه‌ها
- **Levenshtein Distance**: الگوریتم اصلی مقایسه
- **Graphviz**: تولید گراف‌های استاتیک (اختیاری)
- **NumPy**: محاسبات عددی (در صورت نیاز)

## 📦 نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.8 یا بالاتر
- pip (مدیر بسته Python)
- Git (در صورت کلون کردن پروژه)

### مراحل نصب

#### 1. دریافت پروژه
```bash
# کلون کردن از مخزن (در صورت وجود)
git clone <repository-url>
cd textdiff

# یا اگر فایل‌های پروژه را دارید، به دایرکتوری پروژه بروید
cd textdiff
```

#### 2. ایجاد محیط مجازی (توصیه می‌شود)
```bash
# ایجاد محیط مجازی
python -m venv venv

# فعال‌سازی در ویندوز
venv\Scripts\activate

# فعال‌سازی در لینوکس/مک
source venv/bin/activate
```

#### 3. نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

#### 4. تنظیم متغیرهای محیطی
```bash
# کپی کردن فایل نمونه
cp env.example .env

# ویرایش فایل .env و تنظیم SECRET_KEY
# می‌توانید یک کلید مخفی با دستور زیر تولید کنید:
python -c 'from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())'
```

**⚠️ مهم:** متغیر `SECRET_KEY` برای اجرای برنامه ضروری است. حتماً آن را در فایل `.env` تنظیم کنید.

#### 5. اجرای مایگریشن‌های دیتابیس
```bash
python manage.py migrate
```

#### 6. ایجاد ابرکاربر (اختیاری، برای دسترسی به پنل ادمین)
```bash
python manage.py createsuperuser
```

#### 7. راه‌اندازی سرور توسعه
```bash
python manage.py runserver
```

#### 8. دسترسی به برنامه
مرورگر خود را باز کنید و به آدرس `http://127.0.0.1:8000/` بروید.

## 📖 راهنمای استفاده

### رابط کاربری وب

1. به صفحه اصلی در `http://127.0.0.1:8000/` بروید
2. دو متن را در فیلدهای ورودی وارد کنید
3. روی دکمه "مقایسه" کلیک کنید
4. نتایج مقایسه را مشاهده کنید که شامل موارد زیر است:
   - مقایسه کاراکتر به کاراکتر
   - هم‌ترازی کلمه به کلمه
   - درصد شباهت
   - جزئیات عملیات (تطابق، جایگزینی، درج، حذف)
5. گزارش مقایسه را به صورت فایل Markdown دانلود کنید

### API Endpoints

#### مقایسه متون (POST)
```bash
POST /api/compare/
Content-Type: application/x-www-form-urlencoded

text1=سلام&text2=سلام
```

**پاسخ:**
```json
{
    "text1": "سلام",
    "text2": "سلام",
    "highlighted_text1": "...",
    "highlighted_text2": "...",
    "total_cost": 0,
    "operations": [...],
    "similarity": 100.0
}
```

#### دریافت داده گراف (GET)
```bash
GET /api/graph-data/?text1=سلام&text2=سلام&mode=standard
```

**پاسخ:**
```json
{
    "nodes": [...],
    "edges": [...],
    "metadata": {
        "text1": "سلام",
        "text2": "سلام",
        "type": "word_comparison",
        "operations_count": 4,
        "stats": {...}
    }
}
```

## ⚙️ پیکربندی

### متغیرهای محیطی

می‌توانید برنامه را با استفاده از متغیرهای محیطی پیکربندی کنید. یک فایل `.env` در ریشه پروژه ایجاد کنید (می‌توانید از `env.example` به عنوان الگو استفاده کنید):

- `SECRET_KEY`: کلید مخفی Django (**ضروری** - برنامه بدون آن اجرا نمی‌شود)
- `DEBUG`: فعال/غیرفعال کردن حالت دیباگ (پیش‌فرض: True)
- `ALLOWED_HOSTS`: لیست جدا شده با کاما از هاست‌های مجاز
- `MAX_TEXT_LENGTH`: حداکثر طول متن (پیش‌فرض: 10000)
- `MAX_WORDS`: حداکثر تعداد کلمات (پیش‌فرض: 2000)
- `GRAPHVIZ_PATH`: مسیر باینری‌های Graphviz (اختیاری، عمدتاً برای ویندوز)

**مثال فایل `.env`:**
```bash
SECRET_KEY=your-secret-key-here
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1
MAX_TEXT_LENGTH=10000
MAX_WORDS=2000
GRAPHVIZ_PATH=C:\Program Files\Graphviz\bin
```

**نکته:** برنامه به طور خودکار از `python-dotenv` برای بارگذاری متغیرهای محیطی از فایل `.env` استفاده می‌کند.

### حالت‌های مقایسه

- **standard**: مقایسه استاندارد با فاصله لونشتاین

## 🔍 جزئیات فنی

### الگوریتم مقایسه

پروژه از الگوریتم **فاصله لونشتاین** (Levenshtein Distance) استفاده می‌کند که یک الگوریتم پویا (Dynamic Programming) است برای محاسبه حداقل تعداد عملیات ویرایشی مورد نیاز برای تبدیل یک رشته به رشته دیگر.

### پیش‌پردازش متن فارسی

- حذف علائم حرکتی (اعراب) در صورت نیاز
- نرمال‌سازی کاراکترهای فارسی
- تبدیل به نمایش آوایی برای مقایسه آوایی
- مدیریت واو خاموش در کلمات فارسی

### هم‌ترازی کلمات

- استفاده از الگوریتم Needleman-Wunsch برای هم‌ترازی
- محاسبه ماتریس هم‌ترازی
- تعیین مسیر بهینه برای تبدیل

### تجسم

- تولید گراف عملیات ویرایشی
- نمایش ماتریس فاصله ویرایشی
- تجسم تعاملی با D3.js
- تولید تصاویر استاتیک با Graphviz
