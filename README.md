# Text Comparison Web App (Final Project - Data Structures)

This is a Django-based web application for advanced **text comparison and similarity analysis**, developed as a final project for a Data Structures course.

## 🔍 Features

- 🔠 **Standard Text Comparison** (using edit distance)
- 🔊 **Phonetic Comparison** (similar sounding words)
- 📊 **Graph-Based Visualization** of edit operations
- 💡 **Highlighted Differences** in words and sounds
- 🧠 **Custom Algorithm Modes**: `standard`, `phonetic`
- 🌐 **Web UI** for input and comparison
- ⚙️ **RESTful API** for programmatic access
- 📉 **Similarity Score Calculation** (percentage)
- 📁 Support for Persian language

## 🚀 Technologies Used

- Django (Backend Framework)
- HTML/CSS (Frontend templates)
- JavaScript (for UI enhancement)
- Python (Core logic and algorithms)
